📚 教育訓練課程表

共 6 堂課程，建立 AI Scrum Team 全員的 AI 協作核心能力

01
語言模型基礎原理
LLM 是怎麼運作的：Token、Context Window、溫度、幻覺（Hallucination）等核心概念。讓成員理解 AI 的能力邊界，知道什麼情況下可以信任輸出、什麼情況下需要驗證。
全員（6人） 主講：Dimen
02
Context Engineering
如何設計高品質的 Function Context：角色定義、任務背景、輸出格式、限制條件、範例（Few-shot）。這是讓 AI 穩定產出的核心技能，每個成員都必須掌握屬於自己角色的 Context 撰寫方式。
全員（6人） 主講：Dimen
03
Agent Harness 說明
團隊採用的 Agent 框架架構：如何將 AI 接入工作流、工具呼叫（Tool Use）、多步驟任務編排，以及 AI Member 在團隊中扮演的角色與介入時機。
全員（6人）
04
Claude Cowork 使用
Cowork 介面的基本操作：如何建立工作階段、上傳文件、指派任務、查看產出，以及與 AI 進行有效的來回協作。適合所有非工程背景的成員。
PO/BA UIUX SA PMO 非工程角色優先
05
Claude Code
工程成員的核心工具：在終端機或 IDE 中透過 Claude Code 執行程式碼撰寫、重構、測試、Debug。包含如何搭配 CLAUDE.md 設定專案情境，讓 AI 了解 codebase 結構。
AI 工程師 ×2 AI 組長 SA（工程角色）
06
OnePage + Mermaid 顯示技能
如何透過 OnePage（Cloudflare Workers + Pages）對外分享文件，以及使用 Mermaid 語法繪製系統架構圖、流程圖、甘特圖。讓每個角色都能用圖表表達自己負責的領域。
全員（6人）
